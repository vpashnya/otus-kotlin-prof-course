### 1. Управление интеграционными потоками
1.1. **Создание, редактирование, удаление интеграционного потока:**

- **Источник требования:** ИТ-специалисты банка.
- **Описание требования:** Реализовать функциональность создания, редактирования, удаления интеграционного потока. Поля интеграционного потока: 
    - Класс
    - Метод 
    - Включен/Выключен
    - Транспорт (http, kafka, activeMQ и т.д.)
    - Описание интеграционного потока
- **Приоритет требования:** Высокий.

### 2. Управление доступом внешних систем к интеграционным потокам
2.1. **Создание, редактирование, удаление записей о доступе внешней системы:**
- **Источник требования:** ИТ-специалисты банка.
- **Описание требования:** Реализовать функциональность создания, редактирования, удаления записи о доступе внешней системы. Поля записи о доступе внешней системы:
    - Описание внешней системы
    - Идентификатор интеграционного потока
    - Идентифицирующая информация внешней системы (креды, токен и т.д.)
- **Приоритет требования:** Средний.

### 3. Получение информации об интеграционных потоках
3.1. **Представление отображающие информацию об интеграционных потоках:**

- **Источник требования:** Внешние системы пользователи
- **Описание требования:** Реализовать функциональность визуального представления отображающего информацию об интеграционных потоках. Поля представления:
    - Класс
    - Метод
    - Включен/Выключен
    - Транспорт (http, kafka, activeMQ и т.д.)
    - Описание интеграционного потока
    - Внешние системы имеющие доступ к интеграционному потоку
- **Приоритет требования:** Средний.

### 4. Получение уведомление на email при возрастании нагрузки на интеграционном потоке
4.1. **Создание, редактирования, удаления правил уведомления при увеличении нагрузки на интеграционный поток:**
- **Источник требования:** ИТ-специалисты банка.
- **Описание требования:** Реализовать функциональность создания, редактирования, удаления правила об уведомлении. Поля правила об уведомлении:
    - Идентификатор интеграционного потока
    - Количество сообщений
    - Период отслеживания
    - Email для уведомления
- **Приоритет требования:** Средний.
